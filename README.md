# Caesar Cipher Encryption Tool

Task 01 - Cybersecurity Internship (SkillCraft Technology)

## Description
This project implements a Caesar Cipher encryption and decryption program using Python.

The user enters:
- a message
- a shift value

The program encrypts the message and also decrypts it back to the original text.

## Example

Message: Hello  
Shift: 3  

Encrypted Message: Khoor  
Decrypted Message: Hello
